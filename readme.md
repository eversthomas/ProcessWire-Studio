# ProcessWire Studio

Developer toolbox for ProcessWire.

## Features
- Template Code Generator
- Pattern Library
- Field Usage Analyzer
- Development Settings

## Installation
1. Copy to `/site/modules/ProcesswireStudio/`
2. Install via **Modules** page
3. Access via **Setup > ProcessWire Studio**

## Requirements
- ProcessWire 3.0.200+
- PHP 7.4+

## Version
1.0.0 - Initial Release
